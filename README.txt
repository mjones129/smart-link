Smart Link allows you to share single-use links to any page on your site, no passwords required. Create and share links, delete when you're done.

== Description ==

Password-free private link sharing for WordPress&#174;. Protect your client communications with randomly generated access tokens. Each link is unique and can only be used once, ensuring top-notch security.
Links expire after 24 hours. 


== Frequently Asked Questions ==

= How many links can I create? =

There is no limit to how many links you can create.

= Can I create multiple links for the same page? =

Yes you can. This would be useful if you need to share the same page with multiple people.
